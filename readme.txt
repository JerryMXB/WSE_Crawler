0 Run My Crawlers Prerequisite:

1. Python 2.7
2. Beautiful Soup 4 Module
3. html_files/ directory in current directory to store the download pages

========================================================================================================================
1 Run Crawlers:

Python RunBFS.py
Python RunPageRank.py

The default search word is "nyu". You can change the keyword in these two py files, by giving your keyword to initialize
the Crawler. The keywords must by connect with + instead of space.

========================================================================================================================
2 Files:

Readme.txt
Explain.txt - how my crawler work
RunBFS.py - main function for running BFS Crawler
RunPageRank.py - main function for running PageRank Crawler
MyUrl.py - a data structure for storing information related to a url
Crawler.py - a data structure for storing information for one crawling
Auditor.py - a data structure for storing all information about one crawling run
PageRankCrawler.py - PageRank Crawler
BFSCrawler.py - BFS Crawler
